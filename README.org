** How to reproduce

The scripts to build the figures assume the data exist and are
unpacked in [[./data]]. Download from the [[https://doi.org/10.5281/zenodo.14286306][zenodo repository]] the =grid.tar=
and unpack it in =./data=.

#+BEGIN_SRC bash
  cd ./data
  wget -qO- https://zenodo.org/api/records/14286306 | jq -r '.files[].links.self' | xargs -n1 wget -q --content-disposition
  cd ../
#+END_SRC

Run the script [[./build.sh]].
